# Linux Server Health Monitor

A Bash-based Linux monitoring tool that generates a server health report.

## Features

* Hostname Information
* Current User Information
* Date & Time
* System Uptime
* Memory Usage
* Disk Usage
* Top CPU Processes
* Top Memory Processes

## Technologies

* Linux
* Bash Shell Scripting
* Cron Jobs
* Git
* GitHub

## Usage

```bash
chmod +x monitor.sh
./monitor.sh
```
